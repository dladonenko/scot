Aufgaben

Erstellen Sie eine Liste, was Sie von jedem der Punkte erwarten.

Stellen Sie sicher, das es Ziele (!) sind und nicht eine Strategie zur Erreichung Ihrer Ziele.

Seien Sie realistisch. Planen Sie schon im Vorfeld, wie Sie Ihre Position durch gute Gründe, Fakten und Argumente untermauern können.

Definieren Sie immer Ihre „MUSS ICH ERREICHEN" bzw. Ihre „MUSS ICH VERMEIDEN" Position, egal wie unwahrscheinlich es ist, dass dies eintrifft.

Vermeiden Sie stark verflochtene und komplexe „MUSS ICH ERREICHEN" bzw. „MUSS ICH VERMEIDEN" Positionen. Dadurch werden Sie inflexibel.

Testen Sie Ihre MUSS Positionen, indem Sie sich fragen: Wenn Sie nur eine Position auswählen könnten, welche wäre es? Werden dadurch die anderen nicht mehr zum „MUSS"?

Betrachten Sie Ihr MUSS. Warum ist es ein MUSS? Kann das MUSS in einzelne Elemente aufgebrochen werden? Können diese anders definiert werden?

Überprüfen Sie Ihre Ziele immer wieder, bei jeder Gelegenheit.

Wenn Sie nachgeben müssten, können Sie dafür einen „BEABSICHTIGE" oder „MUSS" Punkt erzielen?

Können Sie ein Zugeständnis gegen eine Wunschlistenposition verhandeln?

Können Sie eine neue Forderung gegen eine Wunschlistenposition verhandeln?

Können Sie wichtige Elemente verteidigen indem Sie Ihre Zugeständnisliste benuteen? Könnten Sie so auch eine Sackgasse lösen?
